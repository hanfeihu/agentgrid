# Roadmap

AgentGrid is early, but the direction is intentionally standards-first.

## Now

- Hub, Worker, CLI, MCP server, and Web console
- Node heartbeat and resource reporting
- Node join authorization
- HTTP, command, file, Git, Docker, browser, desktop, plugin, and AgentMessage tasks
- Artifact store and screenshot preview
- Job Runtime with dry-run, shards, checkpoints, and recovery scan
- Tool Registry, Node Tools, and Tool Probe
- OpenAPI and JSON Schema contracts
- English and Chinese documentation

## Near Term

- Placement Engine v2 fully driven by Capability Graph
- Plugin Runtime v1 packaging and installation flow
- Artifact Store v2 large-file backend and log slicing
- richer task detail and evidence timeline UI
- more SDK examples
- MCP tool coverage expansion
- signed Worker update packages

## Ecosystem

- community worker plugins
- hardware bench runbooks
- desktop workbench templates
- browser automation tools
- skill-system integrations
- Open Design-style skill adapters
- mobile console apps
- hosted capability marketplace

## Not In Scope

- natural-language task parsing inside AgentGrid
- replacing AI clients
- replacing remote desktop products
- replacing CI/CD systems
- forcing every node to expose the same capabilities
