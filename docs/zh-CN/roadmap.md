# 路线图

AgentGrid 还处在早期，但方向是标准优先。

## 当前已有

- Hub、Worker、CLI、MCP Server、Web 总控台
- 节点心跳和资源上报
- 节点入网授权
- HTTP、命令、文件、Git、Docker、浏览器、桌面、插件、AgentMessage 任务
- 产物存储和截图预览
- Job Runtime：dry-run、分片、checkpoint、恢复扫描
- Tool Registry、Node Tools、Tool Probe
- OpenAPI 和 JSON Schema
- 中英文文档

## 近期目标

- Placement Engine v2 真正由 Capability Graph 驱动
- Plugin Runtime v1 安装和打包流程
- Artifact Store v2 大文件后端和日志切片
- 更完整的任务详情和证据时间线 UI
- 更多 SDK 示例
- 扩展 MCP 工具覆盖
- Worker 更新包签名

## 生态方向

- 社区 Worker 插件
- 硬件工位 Runbook
- 桌面工位模板
- 浏览器自动化工具
- Skill 系统集成
- Open Design 类 Skill Adapter
- 移动端总控台
- 能力市场

## 不做什么

- AgentGrid 内部解析自然语言任务
- 替代 AI 客户端
- 替代远程桌面产品
- 替代 CI/CD 系统
- 强迫所有节点能力一致
